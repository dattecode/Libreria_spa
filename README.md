# ABP-leonardo-valenzuela 

## Instalación

```sh
npm install
```

### Compilar y Ejecutar para Desarrollo

```sh
npm run dev
```

### Compilar para Producción

```sh
npm run build
```

## Descripción
- El proyecto consiste en un CRUD de libros con categorías y apuntes.
- En el podremos agregar nuestro libros con sus categorias para despues gestionar apuntes en el mismo.
- tenemos 3 paginas la donde crearemos el libro, la donde gestionaremos los libros y la donde gestionaremos los apuntes.# Libreria_spa
